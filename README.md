# Lavalink RTP

A plugin to stream audio from [Lavalink](https://github.com/lavalink-devs/Lavalink) to RTP server.

> [!NOTE]
>
> This plugin is compatible with regular Discord streaming, as long as the same player is not streaming to Discord and
> RTP at the same time.

## Installation

1. Download the latest release from the
   [Releases](https://github.com/Salaron/lavalink-rtp-plugin/releases) page
2. Place `.jar` file to Lavalink plugins directory
